# 📊 European Data Analytics Project

## 🔍 Overview
This project presents an end-to-end data analytics workflow using Python, Jupyter Notebook, and Streamlit. It transforms raw data into meaningful insights through interactive visualizations and analysis.

## 🚀 Key Features
- Interactive Streamlit dashboard
- Dynamic visualizations (bar, line, scatter, histogram, correlation heatmap)
- Real-time data exploration
- Clean and modular project structure

## 📈 Project Impact & Outcome
- Enabled intuitive exploration of complex datasets
- Helped uncover trends, correlations, and anomalies
- Demonstrated ability to convert static analysis into interactive tools
- Showcased end-to-end data analytics skills (EDA → Visualization → Deployment)

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Streamlit
- Scikit-learn
